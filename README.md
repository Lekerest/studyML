### **Lab_2_House_Price_Regression (10.02.2026)**  
Регрессия для предсказания цены дома:

- LinearRegression  
- StandardScaler  
- анализ корреляций  
- отбор признаков  
- Ridge-регрессия  
- подбор параметра `alpha`  
- сравнение метрик: MSE, RMSE, MAE, R²

### **Lab_3_EmbeddingBag_Classification (17.02.2026)**  
Классификация текста (NLP):

- токенизация текста

- биграммы  
- фильтрация редких слов  
- EmbeddingBag  
- линейный классификатор  
- анализ ошибок

---

### **Lab_4_Gradient_Descent (24.02.2026)**  
Регрессия на данных California Housing с использованием PyTorch:

- загрузка данных и удаление цензурированных значений (5.00001)  
- логарифмирование целевой переменной  
- удаление признаков (AveBedrms, Longitude)  
- добавление квадратичных признаков  
- масштабирование (StandardScaler)  
- линейная модель (nn.Linear)  
- оптимизатор Adam с L2-регуляризацией (weight_decay)  
- визуализация процесса обучения  
- анализ остатков (график, гистограмма)  
- корреляционная матрица после преобразований  
- выводы о качестве модели и мультиколлинеарности  

---

### **Lab_5_Classification_and_Regularization (03.03.2026)**  
Классификация и регуляризация на датасетах Iris и Breast Cancer:

- логистическая регрессия  
- L1- и L2-регуляризация  
- сравнение весов модели  
- анализ влияния параметра `C`  
- исследование зануления коэффициентов при L1  
- сравнение распределения весов при L1 и L2  
- работа с multiclass-классификацией через OneVsRestClassifier  
- подбор стратегии для минимизации FN у злокачественных опухолей  
- максимизация Recall для класса `malignant`  
- балансировка классов (`class_weight='balanced'`)  
- подбор `threshold` по `predict_proba()`  
- сравнение модели на всех признаках и на top-10 признаках  
- анализ confusion matrix  
- интерпретация Recall, FN, Accuracy и trade-off между ними
