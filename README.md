## 📂 Структура репозитория

### 🧪 Лабораторные работы

---

### **Lab_1_Data_Preparation (03.02.2026)**  
Предобработка данных:

- train/test split  
- кодирование таргета  
- обработка пропусков (SimpleImputer)  
- масштабирование признаков (MinMaxScaler, StandardScaler)  
- работа с числовыми и категориальными данными  

---

### **Lab_2_House_Price_Regression (10.02.2026)**  
Регрессия для предсказания цены дома:

- LinearRegression  
- StandardScaler  
- анализ корреляций  
- отбор признаков  
- Ridge-регрессия  
- подбор параметра `alpha`  
- сравнение метрик: MSE, RMSE, MAE, R²  

---

### **Lab_3_EmbeddingBag_Classification (17.02.2026)**  
Классификация текста (NLP):

- токенизация текста  
- построение словаря (vocab)  
- биграммы  
- фильтрация редких слов  
- EmbeddingBag  
- линейный классификатор  
- анализ ошибок  
- обучение на датасете AG News (4 класса)

---

### **Lab_4_Gradient_Descent (24.02.2026)**  
Регрессия на данных California Housing с использованием PyTorch:

- загрузка данных и удаление цензурированных значений (5.00001)  
- логарифмирование целевой переменной  
- удаление признаков (AveBedrms, Longitude)  
- добавление квадратичных признаков  
- масштабирование (StandardScaler)  
- линейная модель (nn.Linear)  
- оптимизатор Adam с L2-регуляризацией (weight_decay)  
- визуализация процесса обучения  
- анализ остатков (график, гистограмма)  
- корреляционная матрица после преобразований  
- выводы о качестве модели и мультиколлинеарности
