# 🧠Text Classification with EmbeddingBag  
**Дата:** 17_02_2026  

---

## 🎯 Цель работы

Реализовать модель текстовой классификации на основе `EmbeddingBag`,  
обучить её на датасете **AG News** и проанализировать качество предсказаний.

В рамках лабораторной работы необходимо:

- выполнить токенизацию текста  
- построить словарь (vocab)  
- реализовать модель классификации  
- обучить модель  
- протестировать её на примерах  
- проанализировать ошибки  

---

## 📚 Датасет

Использован датасет **AG News** (HuggingFace).

Количество классов:

- 🌍 **World**
- 🏈 **Sports**
- 💼 **Business**
- 💻 **Sci/Tech**

---

## 🏗 Архитектура модели

Модель реализована по следующей схеме:

Текст → Токенизация → Индексы слов → EmbeddingBag (mean) → Linear → Класс

### Используемые параметры

- Токенизация: `re.findall(r"\w+", text.lower())`
- Добавлены **биграммы**
- Удалены редкие слова (freq ≤ 5)
- Размер эмбеддинга: **128**
- Batch size: **32**
- Оптимизатор: **Adam (lr=0.001)**
- Функция потерь: **CrossEntropyLoss**
- Dropout: **0.5**

---

# 🔬 Эксперименты


## 🔹 1. Базовая модель (без биграмм)

- Только unigram токены  
- 1 эпоха обучения  

**Результат:**
- ~12 ошибок из 20  
- Accuracy ≈ 40–50%

Модель часто путала тексты со смешанной тематикой.

---

## 🔹 2. Улучшенная модель (биграммы + фильтрация редких слов)

Добавлено:

- биграммы  
- удаление редких слов  
- Dropout  

**Результат:**
- 7 ошибок из 20  
- Accuracy ≈ 65%

Качество заметно улучшилось.

---

## 🔹 3. Более строгие параметры (batch_size = 64)

Изменения:

- увеличен размер батча  
- обучение одной полной эпохи  

**Результат:**
- 5–7 ошибок  
- Accuracy ≈ 65–75%

Модель стала стабильнее, однако архитектурные ограничения сохранились.

# ❗ Анализ ошибок

Большинство ошибок возникает в текстах со **смешанной тематикой**:

| Пример текста | Причина |
|---------------|----------|
| Oil prices + Middle East | экономика + политика |
| Apple stocks + iPhone | бизнес + технологии |
| President + Olympic | политика + спорт |
| FIFA corruption | спорт + политика |
| Trade agreement | политика + экономика |

### Причина ошибок

`EmbeddingBag` усредняет эмбеддинги слов и **не учитывает порядок слов**,  
поэтому модель видит набор слов, но не понимает структуру предложения и контекст.

- Реализована модель текстовой классификации на PyTorch.

- Модель корректно классифицирует тексты с явной тематикой.
- Ошибки возникают при пересечении категорий.
- Простая архитектура имеет ограничения и не учитывает порядок слов.
