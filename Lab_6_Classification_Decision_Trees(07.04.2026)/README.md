# Lab 6 — Classification with Decision Trees
07_04_2026

## Цель работы
Исследовать методы классификации на датасете **Rain in Australia** и сравнить качество следующих алгоритмов:
- **C4.5-like** (дерево решений с энтропийным критерием),
- **CART** (дерево решений с критерием Джини),
- **Random Forest**.

В рамках работы была выполнена:
- подготовка данных,
- обработка пропусков,
- кодирование категориальных признаков,
- обучение моделей,
- сравнение качества по метрике **Accuracy**,
- анализ результатов по **precision**, **recall** и **F1-score**.

---

## Используемые метрики

- **Accuracy** — доля правильно классифицированных объектов  
- **Precision** — точность положительного класса  
- **Recall** — полнота положительного класса  
- **F1-score** — гармоническое среднее между precision и recall  

---

# Эксперименты

## 🔹 Exp1 — C4.5-like (Entropy)

В качестве приближения алгоритма **C4.5** использовано дерево решений с критерием **entropy**.

**Результат:**
- Accuracy: 0.7883  

**Classification report:**
- Класс 0: precision = 0.87, recall = 0.86, f1-score = 0.86  
- Класс 1: precision = 0.53, recall = 0.53, f1-score = 0.53  

**Вывод:**  
Модель показала удовлетворительное качество, однако хуже справилась с распознаванием положительного класса, чем случайный лес.

---

## 🔹 Exp2 — CART (Gini)

Для реализации алгоритма **CART** использовано дерево решений с критерием **gini**.

**Результат:**
- Accuracy: 0.7919  

**Classification report:**
- Класс 0: precision = 0.87, recall = 0.86, f1-score = 0.87  
- Класс 1: precision = 0.54, recall = 0.54, f1-score = 0.54  

**Вывод:**  
Алгоритм CART показал немного лучший результат по сравнению с C4.5-like, однако улучшение оказалось незначительным.

---

## 🔹 Exp3 — Random Forest

Для повышения устойчивости классификации использован ансамблевый метод **Random Forest** из 100 деревьев.

**Результат:**
- Accuracy: 0.8572  

**Classification report:**
- Класс 0: precision = 0.87, recall = 0.96, f1-score = 0.91  
- Класс 1: precision = 0.78, recall = 0.50, f1-score = 0.61  

**Вывод:**  
Случайный лес показал наилучший результат по accuracy. Модель существенно лучше определяет класс отсутствия дождя и в целом превосходит одиночные деревья решений.

# Общий вывод

1. Наилучший результат среди исследованных моделей показал **Random Forest** с точностью **0.8572**.
2. Одиночные деревья решений (**C4.5-like** и **CART**) показали близкие результаты, но уступили ансамблевому методу.
3. Алгоритм **CART** незначительно превзошёл **C4.5-like** по accuracy.
4. **Random Forest** лучше обобщает данные за счёт ансамбля деревьев и снижает влияние переобучения.
5. Несмотря на высокую accuracy, для класса дождя (**RainTomorrow = 1**) значение recall остаётся умеренным, поэтому при анализе качества важно учитывать не только accuracy, но и precision, recall и F1-score.
